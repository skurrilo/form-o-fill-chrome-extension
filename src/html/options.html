About Form-O-Fill

        

        

          

            Form-O-Fill is a chrome extension that aims at easing the live of web developers coping with long forms will developing.
          

          

            It was created by Frank Schumacher in one week thanks to the "research week" granted by his employer blau.de.
          

          

            It's source can be found on github.
